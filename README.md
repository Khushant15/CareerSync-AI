# CareerSync AI | Professional Resume & ATS Optimizer

[![Netlify Status](https://img.shields.io/website?url=https%3A%2F%2Faicareersync.netlify.app%2F&label=Netlify)](https://aicareersync.netlify.app/)
[![Render Status](https://img.shields.io/website?url=https%3A%2F%2Fcareersync-ai-rbkp.onrender.com&label=Render)](https://careersync-ai-rbkp.onrender.com)

CareerSync AI is a sophisticated Flask-based web application designed to help job seekers optimize their resumes using hybrid AI analysis. It combines traditional Machine Learning (TF-IDF) with modern LLM-driven insights (Groq/Llama-3) to provide a comprehensive evaluation of career documents.

## 🔗 Live Demo
- **Netlify (Main URL)**: [https://aicareersync.netlify.app/](https://aicareersync.netlify.app/)
- **Render (App Engine)**: [https://careersync-ai-rbkp.onrender.com](https://careersync-ai-rbkp.onrender.com)


## 🚀 Features

- **Hybrid Analysis Engine**: 
    - **TF-IDF Matching**: Mathematical evaluation of semantic overlap between resumes and job descriptions.
    - **Skill Taxonomy**: Automated detection of technical, soft skills, and professional tools.
- **ATS Readiness Audit**: Automated check for common ATS hurdles (complex formatting, low content density, tables).
- **AI-Powered Insights**:
    - **Executive Summary**: A concise professional pitch generated by AI.
    - **Mock Interview Prep**: Tailored technical and behavioral questions.
    - **Drafting Studio**: AI-generated cover letters optimized for specific roles.
- **Dynamic Dashboard**: Interactive visualizations powered by Chart.js for competency profiling.
- **Modern UI**: A premium, "liquid glass" interface designed for maximum user engagement.

## 🛠️ Tech Stack

- **Backend**: Python, Flask
- **AI/ML**: Groq API (Llama-3), Scikit-learn (TF-IDF), NLTK
- **Frontend**: Vanilla Javascript, Chart.js, CSS (Custom Design System)
- **Data Handling**: pdfplumber, python-docx

## ⚙️ Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Khushant15/CareerSync-AI.git
   cd CareerSync-AI
   ```

2. **Set up a virtual environment**:
   ```bash
   python -m venv venv
   source venv/Scripts/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure API Keys**:
   - Create a `.env` file based on `.env.example`.
   - Add your [Groq API Key](https://console.groq.com/) to `config.py` or use environment variables.

5. **Run the application**:
   ```bash
   python app.py
   ```
   The app will be available at `http://127.0.0.1:5000`.
